# Real-Ship Data

This repository is intended to make voyage data collected from a real-ship
data acquisition project, conducted with the support of the Lloyd's Register
Foundation, available to the public.

## About the Data

This project provides voyage data measured on board a 200-meter-class bulk
carrier to the public.

### Collected Data

- **Navigation data** (based on NMEA sentences): key voyage parameters
  including ship position and speed
- **Camera imagery**: video footage of the ship's surroundings
- **Radar imagery**: S-band and X-band radar images
- **Ship motion time series**: roll, pitch, and heave measurements

### Ship Information

- Hull shape of a similar hull form
- Longitudinal mass distribution

## Potential Research Applications

The dataset — combining camera imagery, radar imagery, and motion
measurements — can support research in areas such as:

- Obstacle detection
- Wave environment estimation
- Ship motion prediction

## Access Policy

While this dataset is open to the public, it consists of actual operational
data. To prevent misuse and potential problems, access is granted only after
verification of the requester's identity and purpose.

## How to Request Access

To request access, please send an email to **kim@shippartners.jp** with the
following information:

- Your name and affiliation
- Intended purpose of use

Once your request has been reviewed and approved, FTP access credentials
will be provided.
